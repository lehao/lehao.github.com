服务器性能

May 8th, 2014 | Comments

为什么要去做性能分析呢?

• 降低IT支出 - 找出资源浪费的地区，做出调整，少花钱多办事
• 建立可扩展的体系结构 - 理解系统限制并发展周边系统
• 解决问题 - 找出瓶颈和延迟的异常值

公共的性能指标：

io，cpu，load，rt，qps

分析工具：

基础工具：

• uptime
• top or htop
• mpstat
• iostat
• vmstat
• free
• ping
• nicstat
• dstat

中级工具：

• sar
• netstat
• pidstat
• strace
• tcpdump
• blktrace
• iotop
• slabtop
• sysctl
• /proc

高级工具：

• perf
• DTrace
• SystemTap
• and more ...
